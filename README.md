# ✨ TIL

[📎 노션에서 보기](https://www.notion.so/Today-I-Learn-794830b1e3f34ee29dff5ac2e5b2b51c)

- 흩어져있는 블로그 포스트를 한 곳에 모으기 위해 만든 레파지토리
- **TIL(Today I Learned)** 목적으로 계속 사용


## ✒ 작성 규칙
- 마크다운 파일(.md)로 생성한다.
- 파일명은 영어로 작성한다.
- 루트에 문서를 만들지 않는다.
- 카테고리는 루트의 폴더로 만든다.

## ✒ 글 작성 규칙

```
# 주제

- 카테고리 한글명 (예, 타입스크립트)
- 작성일 (예, 2020년 03월 08일)

## 소제목1
## 소제목2
```
